### Features

- **Flexible Content**: Supports rich HTML content including headings, lists, tables, and links
- **Optional Buttons**: Can display cancel button, confirm button, or both
- **Responsive Design**: Automatically adjusts padding on mobile devices to prevent button overlap
- **Customizable Actions**: Each button can trigger custom callback functions
