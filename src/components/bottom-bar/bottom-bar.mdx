BottomBar is a fixed bottom action area that combines a primary button, optional summary
information, and an optional helper message to guide users through the next step.

#### Params of BottomBar

<ArgTypes of={Stories.WithoutMessage} />

##### Default

<Canvas of={Stories.WithoutMessage} />

##### With message

<Canvas of={Stories.WithMessage} />

#### Usage

<Source
  language="tsx"
  dark
  code={
    `import { BottomBar } from '@components/bottom-bar';

    <BottomBar
      button={{
        text: 'Weiter',
        onClick: () => console.log('Next step'),
      }}
      info={{
        topLeft: 'Monatliche Rate',
        topRight: '39,99 €',
        bottomLeft: 'Laufzeit',
        bottomRight: '24 Monate',
      }}
      message="Sie können den Vorgang jederzeit abbrechen."
    />`

}
