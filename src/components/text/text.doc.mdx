### üìù Typography renderer with optional presets

`Text` renders semantic HTML text with design-system typography tokens. You can configure `tag`, `weight`, `size`, `color`,
and `align` directly, or use a `preset` to apply a predefined combination.

#### Params of Text

<ArgTypes of={Stories.Default} />

#### Usage with `preset`

<Source
  language="tsx"
  code={
    `import { Text } from '@components/text';

    <Text preset="page-title">
      Wellcome to our page
    </Text>`
  }
/>

#### Usage without `preset`

<Source
  language="tsx"
  code={
    `import { Text } from '@components/text';

    <Text
      tag='h1'
      lined
      align='left'
      weight='regular'
      size='hl1'
      color='success'
    >
      Wellcome to our page
    </Text>`
  }
/>
