#### Implements interfaces:

- `Interactive<string>`: This interface includes `id`, `value`, `disabled`, `onChange`, `onFocus`, `onBlur`.
- `Base`: This interface includes basic properties such as `className` and `data-testid`.

#### Description

`ControlInput` renders a single-line text input. Supports optional date masking `TT/MM/JJJJ` style or
`text` input. This is a controlled component, so the value is managed by the parent component and passed
down via props. This component is designed to be flexible and can be used in various contexts where user
input is required.
