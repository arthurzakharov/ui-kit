<Subtitle>⚠️ Error text indicator for form validation</Subtitle>

##

`ControlErrorMessage` renders validation feedback text in an error style. Use it below a control to communicate required fields, invalid input, or any form-level validation issue.

#### Example

<Canvas of={Stories.Default} />

#### Usage

<Source
  language="tsx"
  code={
    `import { ControlErrorMessage } from '@controls/control-error-message';

    <ControlErrorMessage>
      This field is required
    </ControlErrorMessage>`

}
