`ControlRadioText` renders styled text labels for radio buttons and checkboxes. Use it to display option text with
consistent sizing, coloring, and checked/unchecked states. Supports HTML content and optional single-line truncation.

#### Params of <b>ControlRadioText</b>

<Controls of={stories.SizeBody} />

#### Interactive Example

<Canvas of={stories.ColorPrimary} />

#### Usage

<Source
  language="tsx"
  dark
  code={
    `import { ControlRadioText } from '@controls/control-radio-text';

    <ControlRadioText
      size="body"
      color="text-primary"
      checked={false}
      oneLine={false}
      className="custom-class-name"
    >
      Lorem ipsum dolor sit amet
    </ControlRadioText>`

}
/>
